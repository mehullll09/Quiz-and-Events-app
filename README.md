#  Django Quiz_events

A full-stack web application built with **Django** and **Tailwind CSS** that allows users to:
- Take interactive quizzes and get instant results.
- View upcoming events.
- Manage all data dynamically using Djangoâ€™s ORM and admin panel.

This project demonstrates backend and frontend integration with clean design and efficient data handling using Django relationships and templates.

---

##  Features

###  Quizzes Module
- List available quizzes.
- Attempt multiple-choice questions (MCQs).
- Auto-evaluate answers and display score instantly.
- Store user submissions and results in the database.

###  Events Module
- Display upcoming events with details like title, date, and location.
- Organized layout using TailwindCSS.

###  Admin Functionality
- Create and manage quizzes, questions, answers, and events easily via Django Admin.
- Database-driven structure using Django ORM.

###  Additional Highlights
- Optimized with `prefetch_related()` for efficient query loading.
- Responsive UI built with TailwindCSS CDN.
- Easy setup and clean code structure.
